🚀 3 步快速启动（5 分钟）：

1️⃣ 启动后端（终端 1）
   $ cd backend
   $ python -m venv venv
   $ source venv/bin/activate
   $ pip install -r requirements.txt
   $ python main.py
   ✅ http://localhost:8102

2️⃣ 启动后台管理（终端 2）
   $ cd frontend
   $ npm install
   $ npm run dev
   ✅ http://localhost:3000

3️⃣ 启动博客门户（终端 3）
   $ cd portal
   $ npm install
   $ npm run dev
   ✅ http://localhost:5173

❓ 常见问题快速解答：

Q: 端口被占用怎么办？
A: kill -9 <PID> (macOS/Linux)
   taskkill /PID <PID> /F (Windows)

Q: npm install 失败？
A: npm cache clean --force
   npm config set registry https://registry.npmmirror.com

Q: 前端连接不到后端？
A: 确保后端运行在 8102 端口
   检查代理配置
   查看浏览器 DevTools Network

Q: 如何在两个前端间导航？
A: 在后台添加链接到 http://localhost:5173
   在门户添加链接到 http://localhost:3000
