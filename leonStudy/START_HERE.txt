   后端：
   $ cd backend
   $ python -m venv venv
   $ source venv/bin/activate     (macOS/Linux)
   $ pip install -r requirements.txt
   $ python main.py

   前端：
   $ cd frontend
   $ npm install
   $ npm run dev
